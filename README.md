# Kalendarzyk

Tracker cyklu menstruacyjnego — Flutter / Android. Notuj datę pierwszego dnia menstruacji, przeglądaj cykle w kalendarzu, czytaj prognozę kolejnego cyklu (średnia z ostatnich 6 cykli).

Dane trzymane lokalnie w Hive (`box_cycles`), bez chmury. Eksport/import JSON w bocznym menu.
